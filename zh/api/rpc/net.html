# net_peersCount

返回网络中的节点个数

  • Parameters: 无

  • Returns:

    • peersCount
      • all : 网络所有的节点个数
      • connect : 与当前节点建立连接的节点个数
      • online : 网络中所有在线节点

  • Example:

{
  "jsonrpc": "2.0",
  "id":2,
  "method":"net_peersCount"
}
{
  "jsonrpc": "2.0",
  "id": 18,
  "result": {
    "all": 3,
    "connect": 2,
    "online": 3
  }
}
{
  "jsonrpc": "2.0",
  "id":2,
  "method":"net_peersCount"
}

# net_syncing

返回节点同步状态

  • Parameters: null
  • Returns:
    • 连接同步状态：true 代表正在同步，false 代表没在同步
  • Example:
{
  "jsonrpc": "2.0",
  "id":3,
  "method":"net_syncing"
}

{
	"jsonrpc": "2.0",
	"id": 3,
	"result": false
}

{
  "jsonrpc": "2.0",
  "id":3,
  "method":"net_syncing"
}

# net_getBandwidthStats

返回带宽指标

  • Parameters: null
  • Returns:
    • 带宽状态 :
      • TotalIn TotalOut ： TotalIn和TotalOut字段记录已发送/已接收的累积字节。
      • RateIn RateOut ： RateIn和RateOut字段记录每秒发送/接收的字节数。
  • Example:
{
	"jsonrpc": "2.0",
	"id":18,
  	"method":"net_getBandwidthStats"
}
{
  "jsonrpc": "2.0",
  "id": 18,
  "result": {
    "TotalIn": 598546022,
    "TotalOut": 3524381031,
    "RateIn": 7498.377919815488,
    "RateOut": 28352.544366668382
  }
}
{
	"jsonrpc": "2.0",
  	"id":18,
  	"method":"net_getBandwidthStats"
}
