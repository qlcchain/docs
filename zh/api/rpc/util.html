# util_rawToBalance

返回最小单位余额对应的指定单位的余额

  • Parameters:

    • string : 最小单位的余额
    • string : 单位，如果 token 是 QLC，可设置为 qlc , Kqlc , QLC 或者 MQLC , 其他 token 设置为空字符串
    • string : token 名称，可空，如果不设置，默认为 QLC

  • Returns:

    • string : 指定单位的余额

  • Example:

{
  "jsonrpc": "2.0",
  "id":1,
  "method":"util_rawToBalance",
  "params":  ["200000000","QLC"]
}

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "2"
}

{
  "jsonrpc": "2.0",
  "id":1,
  "method":"util_rawToBalance",
  "params":  ["200000000","QLC"]
}
