# ptmkey_getPtmKeyByAccount

Get a ptm pubkey by account

  • Parameters:

    • account : target account

  • Returns:

    • account: target account

  • btype: business type

  • pubkey: ptm pubkey

  • Example:

{
	"jsonrpc": "2.0",
	"id": 3,
	"method": "ptmkey_getPtmKeyByAccount",
	"params": [{
	    "qlc_1s77torrs38rkm4eh7z5rxei3sqsdfjq9am93cjuf15h9y3cjyq8tf6u8d7j"
	}]
}
{
